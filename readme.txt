USER ROLES & PERMISSIONS IN EXAMSYS
ADMINISTRATOR ROLE
Dashboard Access
✅ View system statistics (total students, teachers, courses, marks entered)
✅ Quick access buttons to all management modules
✅ System information panel
Student Management
✅ Add new students with complete profile (roll number, name, program, department, semester, batch year, contact, email)
✅ Edit existing student information
✅ Delete students (with confirmation dialog)
✅ View all students in paginated table
✅ Search students by roll number or name
✅ View detailed student profiles
Teacher Management
✅ Add new teachers with profile (employee ID, name, department, designation, contact)
✅ Edit teacher information
✅ Delete teachers (with confirmation)
✅ View all teachers in paginated table
✅ Search teachers by name or employee ID
Course Management
✅ Create courses (course code, name, credits, department, semester)
✅ Edit course details
✅ Delete courses (with confirmation)
✅ View course catalog organized by department/semester
Course Assignment
✅ Assign teachers to courses for specific semesters/batches
✅ View all course assignments
✅ Unassign teachers from courses
✅ Prevent duplicate assignments (same teacher + course + semester)
Exam Management
✅ Create exam sessions (name, type, semester, batch year, start/end dates)
✅ Edit exam details
✅ Delete exams (with confirmation)
✅ Set exam status (Draft/Published)
✅ Publish/Unpublish results with confirmation dialog
✅ View all exams with status indicators
Student Enrollment
✅ Enroll students in courses for specific semesters
✅ View all enrollments
✅ Remove students from courses
✅ Manage batch-wise enrollments
Mark Entry & Management
✅ Enter marks for any course/exam (internal + external)
✅ Edit existing marks
✅ Delete marks (with confirmation)
✅ View all marks across all courses/exams
✅ Override teacher-entered marks (with audit trail)
✅ Bulk mark entry for multiple students
Result Generation
✅ View all results before publication
✅ Generate grade distribution charts (bar charts)
✅ Calculate CGPA for any student/exam
✅ Publish results for student viewing
✅ Unpublish results (hide from students)
✅ View pass/fail statistics per exam
✅ View average marks per exam
System Administration
✅ Manage grading scale configuration
✅ View audit trail logs (all critical operations)
✅ Export data to CSV (future enhancement)
✅ Backup database
✅ System settings management
TEACHER ROLE
Dashboard Access
✅ View assigned courses count
✅ View marks entered count
✅ Quick access to mark entry system
✅ View list of assigned courses with semester/batch details
Course Access
✅ View only courses assigned to them
✅ Cannot view courses assigned to other teachers
✅ Cannot create/edit/delete courses
✅ Cannot assign/unassign themselves to courses
Mark Entry
✅ Enter marks for students in their assigned courses only
✅ Enter internal marks (0-40) and external marks (0-60)
✅ View real-time grade calculation during entry
✅ Mark students as UFM (Unfair Means)
✅ Save marks with transaction safety
✅ View previously entered marks
✅ Edit their own entered marks (before publication)
✅ Cannot enter marks for courses not assigned to them
✅ Cannot publish results (admin only)
✅ Cannot delete marks (only admin can)
Result Viewing
✅ View marks they have entered
✅ View grade distribution for their courses (after publication)
✅ View student performance statistics
✅ Cannot view unpublished results
✅ Cannot view marks entered by other teachers
Limitations
❌ Cannot manage students
❌ Cannot manage teachers
❌ Cannot create/edit exams
❌ Cannot publish/unpublish results
❌ Cannot access audit logs
❌ Cannot view system statistics
STUDENT ROLE
Dashboard Access
✅ View personal profile information (name, roll number, program, department, semester, batch year)
✅ View enrolled courses count
✅ View published results count
✅ Quick access to results and hall ticket
Result Viewing
✅ View published results only (after admin publishes)
✅ View detailed grade breakdown (internal/external/total marks)
✅ View grade and grade point for each course
✅ View CGPA calculation with formula
✅ View grade scale legend
✅ View pass/fail status
✅ Cannot view unpublished results (shows "Results Not Published" message)
✅ Cannot view results of other students
✅ Cannot edit or modify any marks
Hall Ticket Access
✅ View hall ticket with student details
✅ Download hall ticket (print functionality)
✅ Print hall ticket with proper formatting
✅ View exam schedule and dates
✅ Cannot modify hall ticket information
Exam Information
✅ View published exam schedules
✅ View exam dates and types (midterm/final/quiz)
✅ View semester and batch year information
✅ Cannot view unpublished exam schedules
Limitations
❌ Cannot enter or edit marks
❌ Cannot view other students' results
❌ Cannot access mark entry system
❌ Cannot view unpublished results
❌ Cannot manage courses or enrollments
❌ Cannot access admin/teacher dashboards
❌ Read-only access to all features
